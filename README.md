# Medical Equipment Usage Analysis

🔗 **Live Dashboard:** https://medical-equipment-usage-analysis.streamlit.app/

This project analyzes hospital equipment usage to identify patterns, optimize utilization, and reduce costs.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Peak usage hours identified
- Cost-heavy equipment detected
- Maintenance impact analyzed
